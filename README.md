# 快消品电商海报生成器

这是一个基于 React 和 Vite 构建的现代化 Web 应用，旨在帮助快消品电商运营人员快速、高效地生成专业、精美的商品宣传海报。

本工具深度优化了批量操作流程，支持从 Excel 文件或表格复制内容直接导入商品信息，并提供多种预设模板和设计风格，让海报制作变得前所未有的简单。

## ✨ 核心功能

- **多种布局模板**：内置多种商品布局模板（单商品、2x2、2x3、3x3等），满足不同数量商品的展示需求。
- **专业设计风格**：提供多种预设的设计风格（如专业商务、清新活力、温馨家居等），一键切换海报整体视觉效果。
- **高度自定义**：支持自定义海报的几乎所有元素，包括店铺名称、宣传标语、主色调、宣传图、底部文字和二维码。
- **实时预览**：所有配置的更改都会实时反映在预览区域，所见即所得。
- **强大的批量导入**：
  - **支持从 Excel 文件导入**：可直接选择 `.xlsx` 或 `.xls` 文件，程序会自动解析并填充商品信息。
  - **支持从表格粘贴**：可直接从 Excel、Numbers 等电子表格软件中复制多行多列数据，并粘贴到导入框中。
  - **智能容错**：导入功能足够健壮，能够兼容空值（如无生产日期），并自动调整海报布局，确保美观。
- **一键下载**：完成设计后，可一键将高质量的海报图片下载到本地。

## 🚀 技术栈

- **前端框架**: React
- **构建工具**: Vite
- **UI 组件库**: shadcn/ui
- **Excel 文件解析**: SheetJS (xlsx)
- **核心绘图**: HTML5 Canvas

## 快速开始

请按照以下步骤在本地运行本项目：

1.  **克隆仓库**
    ```bash
    git clone [你的仓库地址]
    cd poster-generator-v32
    ```

2.  **安装依赖**
    推荐使用 `pnpm` 进行安装：
    ```bash
    pnpm install
    ```
    如果 `pnpm` 不可用，也可以使用 `npm` 或 `yarn`。

3.  **启动开发服务器**
    ```bash
    pnpm run dev
    ```
    启动成功后，在浏览器中打开 `http://localhost:5173` 即可访问。

## 📋 批量导入说明

为了获得最佳的批量导入体验，请确保您的数据格式符合以下规范。

### 数据格式

无论是通过 Excel 文件导入，还是直接粘贴，都请遵循以下列顺序和标题（程序会进行模糊匹配）：

| 商品标题 (必填) | 商品规格 (必填) | 生产日期 (可选) | 划线价 (可选) | 促销价 (必填) | 商品图片链接 (可选) |
| :---------- | :---------- | :---------- | :---------- | :---------- | :---------- |
| 优质苹果      | 500g/袋     | 2024-07-15  | 29.90       | 19.90       | https://.../apple.jpg |
| 新鲜香蕉      | 1kg/串      | 2024-07-14  | 12.50       | 9.90        | https://.../banana.jpg |

- **从表格粘贴**：直接在 Excel 或其他表格软件中选中数据区域（**包含标题行**），复制，然后粘贴到“数据预览与编辑”文本框中即可。程序使用**制表符（Tab）**作为分隔符。
- **从文件导入**：点击“从Excel导入”按钮，选择您的文件。请确保数据在第一个工作表（Sheet）中。
